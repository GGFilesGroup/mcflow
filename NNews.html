The Frostfire Update (10th July, 2025)
EaglercraftX (Eaglercraft 1.12.2) has officially launched! This update brings major performance improvements, new server features, and community tools. EaglercraftX now supports 1.12.2 with full compatibility, allowing smoother gameplay, enhanced multiplayer stability, and fresh content mods. Jump in and enjoy the improved experience with your friends!

  • Added 1.12.2 servers to the official server list
  • Introduced dynamic weather effects on the website
  • Added support for custom resource packs in EaglercraftX
  • Improved server list UI and responsiveness
  • Updated EaglercraftX core to latest stable build
  • Fixed multiple crash bugs and memory leaks
  • Removed deprecated legacy features to boost performance

The Bountiful Update (2nd December, 2022)
Originally released on July 28, 2015
Eaglercraft has been updated to Minecraft version 1.8.8 to bring a bounty of new things to Minecraft. Switching to 1.8.8 is highly recommended from older versions as it introduces a ton of new features, stability improvements, and better mod support.
